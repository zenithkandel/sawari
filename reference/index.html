How It Works

Three simple components working together to bridge the connectivity gap.

01

Spark Node

A rugged ESP32 + LoRa device worn by trekkers and installed at checkpoints. One button press sends an encoded SOS signal that consumes minimal bandwidth.

  • 7-day battery life
  • Works in -20°C
  • Waterproof design

Base Station

Technology

Built with proven, open technologies optimized for extreme conditions.

LoRa Protocol

Long Range, Low Power radio technology capable of 15km+ transmission in mountain terrain.

ESP32 Hardware

Dual-core microcontroller with WiFi and Bluetooth, designed for IoT edge computing.

Long Range Radio

LoRa technology enables 15km+ direct transmission from remote client nodes to base stations without repeaters.

Index Mapping

Bandwidth optimization system that encodes messages as integers, minimizing transmission size.

FCM + Email

Firebase Cloud Messaging for instant push notifications plus automated email alerts.

PWA Portal

Progressive Web App dashboard that works offline and installs on any device.
